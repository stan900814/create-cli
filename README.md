# y-cli

## 基于vite快速构建项目脚手架

### 1. 环境依赖
> [Node.js](https://nodejs.org/en/) version > 14.18, < 16+
> [Vite](https://vitejs.dev/) 

### 2.使用说明

```bash
$ npm install @stan900814/create-cli -g
```

if success,then

```bash
$ create-project
```
then,finally

```bash
cd my-project

npm install
npm run dev
```
